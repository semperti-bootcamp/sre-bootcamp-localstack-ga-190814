## Generar S3 Bucket con Terraform

```
cd terraform/s3
terraform init
terraform apply
```

Checkear el s3 bucket creado

aws s3 ls s3://NAMEBUCKET

## Generar la tabla DynamoDB con Terraform

```
cd terraform/dynamodb
terraform init
terraform apply
```

Listar las tablas
```
aws dynamodb list-tables
```
